 Le Donjon Auto‑Conscient
Genre : expérimental / IA‑centric
Pitch :
Un donjon qui s’auto‑adapte aux joueurs : il apprend leurs stratégies, modifie ses pièges et réécrit son propre lore.
Meta‑jeu :

Chaque agent représente un “sous‑système” du donjon,
Le MJ n’est plus omniscient mais chef d’orchestre,
Les joueurs peuvent tenter de parler au donjon.

🎯 Intérêt pédagogique : émergence, orchestration, limites du contrôle.


AI gamemaster call with MCP a Dice roll server
Then a call to rule agent in a knowledge base
Then a call to a character agent stored in a character database
