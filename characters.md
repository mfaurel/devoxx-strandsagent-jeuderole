# Personnages Joueurs — Le Donjon Auto‑Conscient

> Fiches prêtes à l'emploi pour le workshop. 4 personnages pré-générés + règles de création rapide.

---

## Règles de Création Rapide

1. Choisir une **classe** (voir ci-dessous)
2. Répartir **72 points** entre FOR / DEX / CON / INT / SAG / CHA (min 8, max 18)
3. Calculer les **modificateurs** : `(stat - 10) / 2` arrondi inférieur
4. Calculer les **PV** : `10 + mod CON`
5. Choisir **3 compétences**
6. Choisir un **nom** et un **trait de personnalité** (1 phrase)

---

## Classes Disponibles

| Classe | Stat principale | Spécialité |
|---|---|---|
| **Guerrier** | FOR | Combat, résistance, second souffle |
| **Roublard** | DEX | Discrétion, pièges, attaque sournoise |
| **Mage** | INT | Sorts offensifs, déduction, arcanes |
| **Clerc** | SAG | Soins, protection, résistance au donjon |

### Personnage 1 — Kira la Rapide
**Classe :** Roublarde  
**Concept :** Ancienne cartographe de la guilde des Cartographes de l'Inutile. Elle a tenté de cartographier l'Œil de Basalte une fois. Sa carte est célèbre pour avoir 47 corrections et un mot final : *"Non."*

| Stat | Valeur | Mod |
|---|---|---|
| FOR | 10 | +0 |
| DEX | 18 | +4 |
| CON | 12 | +1 |
| INT | 14 | +2 |
| SAG | 13 | +1 |
| CHA | 9 | -1 |

**PV :** 11  
**CA :** 14 (armure légère)  
**Compétences :** Discrétion, Investigation, Outils de voleur  
**Arme :** Dague (1d4+4) × 2 attaques par tour  
**Capacité spéciale — Attaque sournoise :** Si avantage ou allié adjacent, +2d6 dégâts  
**Ressource :** 3 points d'astuce / session (récup : repos court)

**Trait :** "Je ne dis jamais que quelque chose est impossible. Je dis que ça coûte cher."

**Note pour le donjon :** Le Stratège la classifie comme *"imprévisible — données insuffisantes"*. Le Chronicler la nomme *"Celle Qui Refuse Les Cartes"*.

## Note sur l'Évolution des Personnages

Le donjon *apprend* les personnages autant que les joueurs apprennent le donjon.

Après chaque session significative, le Chronicler génère une **"fiche donjon"** pour chaque PJ :
- Son pattern tactique principal
- Son nom dans le lore
- Ce que le donjon a prévu pour lui

Ces informations ne sont pas données aux joueurs — elles guident les adaptations des autres agents.
